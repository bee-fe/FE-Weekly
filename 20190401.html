# BigPipe的应用 (opens new window)

  • 推荐理由: 使用 bigpipe 方案，减少首屏显示时间，提高大型复杂页面的性能加速，提升用户体验。在 node 中间层的基础上，实现 bigpipe 方案，通过 node 层请求多个接口，获取数据，然后分块传输给客户端，可以有效地解决页面出现大量 api 请求，消耗网络传输时间，以及并发限制带来的请求阻塞和滞后等问题。
  • 推荐人: Allen Xiang
