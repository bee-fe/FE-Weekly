# 2019年前端面试都聊啥？一起来看看 (opens new window)

  • 推荐理由: 要使一场面试成功，必须对其进行讨论。它必须是一个交换思想的地方。它应该引发人们思考并客观地分析给定的问题。它要能帮助我们理解人们做决策的过程。它要能帮助我们了解一个人对技术和解决问题的热情。它意味着要了解未来可能的同事。所有那些谜题，tricks 或 typeof null 都不能真正称为面试。
