# CSS 与网络性能 (opens new window)

  • 推荐理由: CSS 是页面渲染的关键因素之一，（当页面存在外链 CSS 时，）浏览器会等待全部的 CSS 下载及解析完成后再渲染页面。关键路径上的任何延迟都会影响首屏时间，因而我们需要尽快地将 CSS 传输到用户的设备，否则，（在页面渲染之前，）用户只能看到一个空白的屏幕。那么，最大的问题是什么？有什么好的解决办法？这篇文章会有着详细的介绍。
  • 推荐人: 曹菁
